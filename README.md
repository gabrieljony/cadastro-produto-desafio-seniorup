# store-computers

> Store Computers - Loja de Cadastro de Computadores com suas Categorias

## Build Setup

Back-End
``` bash
# serve with hot reload at localhost:5001
dotnet run

```

Front-End

``` bash
# install dependencies /clientapp
npm install

# serve with hot reload at localhost:8081
npm run dev

```
## Tecnologias usadas
